# Websockets chat

I decided to get familiar with websockets in [Nest.js](https://github.com/nestjs/nest), realtime chat is the first thing that came to my mind

## Features

- PassportJS/JWT auth
- Rooms
- Kick/Ban user

## Installation

```bash
$ npm install
```

## Running the app

```bash
# start docker containers
$ docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev
```
